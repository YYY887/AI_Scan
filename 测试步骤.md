# 测试步骤

## 1. 重新加载插件
1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 找到"AI截图分析助手"插件
4. 点击刷新按钮 🔄（或者先删除再重新加载）

## 2. 测试截图功能
1. 打开任意普通网页（如 baidu.com、github.com 等）
2. 点击浏览器工具栏的插件图标
3. 点击"开始截图"按钮
4. 应该看到网页变暗，鼠标变成十字光标
5. 拖动鼠标框选区域
6. 松开鼠标后应该弹出对话框

## 3. 查看调试信息
如果还是没反应，按 F12 打开开发者工具：

### 查看 Console 标签
应该能看到：
- "截图扩展初始化"
- "收到消息: {action: 'startCapture'}"
- "开始截图模式"

### 查看 Background Service Worker
1. 在 `chrome://extensions/` 页面
2. 找到插件，点击"service worker"链接
3. 查看是否有错误信息

## 4. 常见问题

### 问题1: 点击没反应
- 确保不是在 chrome:// 或 edge:// 等特殊页面
- 刷新当前网页后再试
- 重新加载插件

### 问题2: 提示权限错误
- 检查 manifest.json 中的 permissions
- 重新加载插件

### 问题3: 截图后没有对话框
- 按 F12 查看 Console 是否有错误
- 检查 styles.css 是否正确加载

## 5. 测试 API 调用
1. 截图成功后，在对话框中填写：
   - API Key: 你的豆包 API Key
   - 模型ID: 你的模型端点ID
2. 点击"开始分析"
3. 应该显示"正在分析中，请稍候..."
4. 等待几秒后显示分析结果
