# 图标文件

请在此文件夹中放置以下尺寸的图标：

- icon16.png (16x16像素)
- icon48.png (48x48像素)  
- icon128.png (128x128像素)

## 快速生成图标

你可以使用以下方式生成图标：

1. **在线工具**：
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/

2. **使用Emoji**：
   - 访问 https://favicon.io/emoji-favicons/
   - 选择一个emoji（如📸、🤖、✂️）
   - 下载并重命名为对应尺寸

3. **自己设计**：
   - 使用Figma、Photoshop等工具设计
   - 导出为PNG格式，分别保存为不同尺寸

## 临时方案

如果暂时没有图标，可以使用纯色占位图：
- 创建一个简单的PNG文件
- 背景色可以用蓝色 (#1a73e8)
- 中间放一个白色的相机或截图图标
